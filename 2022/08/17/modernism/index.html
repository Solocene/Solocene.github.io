现代性与我

本文最后更新于：2022年8月19日 晚上

现在我正坐在内江开往成都的高铁上。我不会在那里停下，我将继续穿过川北丘陵、汉中盆地、秦岭、黄河、华北平原，然后到北京西站。这趟旅程长达10个小时，我觉得不能辜负车窗外的风景，便想着写下这篇blog。

这个暑假，我接触了太多新东西和新概念，有了太多新感触，一开始我还会把它们记下来，后来也不记了，看了就忘掉，只在记忆中留下一道划痕。所以为什么我想到要以“现代性”为主题呢？

不为什么，随便想的。随便看看就好了。

俄狄浦斯与那尔喀索斯

Narcissus(N) 哇，多美妙的世界，多完美的人！

N 万物之灵长，世界之主宰！人啊，我赞美你！

N 直尺和天平面前，王冠和权杖显得如此可笑！理性的光芒啊，你铸成了人的荣耀！

Oedipus(O) 吾王荣耀，不得玷污！吾王蒙天父圣恩方执权杖，你又是何人，竟敢造次！

N 可笑，可笑。俄狄浦斯，你认错了父！给你吃食的是厨师，不是天父；给你住处的是房东，不是天父； 给你快乐的是我们的通俗作家，不是天父！

O 你这一说，也有几分道理。但你，天天在报纸上写的什么东西？我觉得我们国王挺好的，你为什么想推翻它？

N 你只看得见国王，却看不见你的邻居！楼下老太太的儿子在炼钢厂干活，失去了一个手臂；对面的女孩为了买一件首饰，跑到拉丁区去站街；街角的男人得了肺痨，钱却都被酒吧老板骗走了……这些事在我看来都难以忍受！而你，就像听故事一样听着这些，却对你从没见过的国王感恩戴德！

O 有道理。我也是一个市民。

O 我爱我的社区，我关注本地新闻，我爱我生活的地方！毕竟我属于他们。

N 还玩儿这些呢，这些早就过时啦！你只关注你们那个小城市的新闻，家里还铺着你奶奶留下来的地毯，喜欢去剧院听歌剧，听哪些几百年前写的故事。呸！你还不知道吗？现在每天都会出现一个新的科技，新的发现！你应该把眼界放开阔点儿……堪萨斯城有个人发明了柴油机飞艇！波兰在入侵卢森堡！奥地利的工人又起义了！一个俄国人说他发现了崭新的艺术……嘿，老俄，这才是现代人该看的东西！世界正在重组，历史马上又会到达一个关键的节点！别落下了！

O 好吧，我是有点跟不上时代了。

O 嘿老弟，我现在每天晚上定时看电视新闻，每个月看一场好莱坞电影，每周都看时尚杂志！疯狂星期一、黑色星期二，我一个都不落下！听最流行的音乐、穿最潮的搭配。我的思想是最主流的，我的生活是最时髦的！

N 呵呵。不错，这次你们人确实多了。不过这依然掩盖不了你们是被商业文化和资本主义政府催眠成的行尸走肉这一事实！资本吹嘘什么，你们就追捧什么；政府说我们民主，你们就真以为投个票就民主了；大报纸、大电视台，让你们以为自己耳听八方，实则在愚弄你们！看看吧，异族依然被歧视！女性还被当玩物！核弹还悬在我们的头上，工业废水和杀虫剂正在毒害我们的土地！你以为你什么都可以知道，但你什么都不知道！快跟我一起扬起反抗的旗帜，向这个无形的暴君开战吧！

O 喂，这回我绝对不会跟你们疯了。你们这帮小年轻，成天就知道反反反，我问你，反完你们准备建一个怎么样的社会？现在我们吃饱穿暖，还有这么多好玩的，有什么不好的？叫你说说纲领你还说不出来，我票都没法投给你，你准备怎么反？一起建设自由民主/社会主义，不好吗？

N 呸。

O 不过你说了几个问题还是挺好的，确实该解决解决。你看，现在大家都不准种族歧视了，女权也挺红火。各种禁止核试验条约已经签订，环保也成了政治正确。听说你在玩摇滚？我适应适应还觉得挺好听的，现在迪厅都有好多摇滚乐哦。你看，反啥，不搞反也可以解决嘛。

N 呸呸呸！这根本不叫解决……那个无形的，企图影响所有人的，还没有被解决！你们还是沉浸在那种……哈哈，还自由民主/社会主义，你自己信这玩意吗？

O 你话都说不利索了。不过，我确实不怎么信那种宏大叙事了。确实，嗯。不过那又怎样？我现在只希望我自己能过的更好。这还不是跟你学的？现在你搞的那些玩意，说实话看不懂。算了吧，现在你已经没得反了。我说你们这些小年轻怎么就不信邪呢。

N 呸！你们**的就是***。不管你们了，我去酒吧了。

N后来怎么样，确实不知道。有人说去搞互联网了，混的还不错，也成了ACG爱好者。有人说进了一个天坑专业，出来干了完全不同的一行，平时喜欢搞搞电音。有人说进了富士康的一个厂，一个月出来一两次，偶尔在抖音上发发生活视频和一些牢骚，前段时间号又被封了。有人说的就有点玄乎了，说他之后还在搞朋克，还染上了毒，有一天凌晨在酒吧唱完到池塘边走路，看着自己在水里的倒影发笑，笑着笑着就栽下去了。几年过去了，听说他落下去的那个池塘长了很多水仙花。

O也过得没有多好。自从N失踪后，O经常半夜醒来，满头大汗而且泪流满面。O再也没有做过梦了。O开始变得消沉，经常不去工作被老板辞退了。O每天晚上都会抑制不住的哭，时间长了，竟然把两只眼睛哭失明了。O说自己估计活不了多少时日了。
